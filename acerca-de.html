    Sobre este Proyecto
    • Descripción General

      Este sitio web contiene tres proyectos los cuales abarcan todos los conocimientos vistos a lo largo de la materia.

      Los diferentes proyectos están realizados en HTML5, CSS3, JavaScript y consumen API's, además de estar subidos en Dockers dentro de la nube de Google.

    • Calculadora JS

      Esta es una calculadora simple hecha con HTML5, CSS3 y JavaScript, puede hacer operaciones básicas (+ - * /).

    • API Clima

      Este es un pequeño form hecho con HTML5, CSS3 y JavaScript que consume la API de "openweathermap.org" y con base en una latitud y longitud, despliega la ciudad y temperatura.

    • API LoL

      Este es un pequeño form hecho con HTML5, CSS3 y JavaScript que consume la API de "developer.riotgames.com" y con base en un nombre de invocador despliega el nivel del jugador y su ícono dentro del juego.

    • Contacto

      Este es un formulario de contacto hecho con HTML5, CSS3 y JavaScript el cual solo es estético, no funcional.
